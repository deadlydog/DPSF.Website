DPSF Support (Dynamic Particle System Framework)

Support / Comments for DPSF

DPSF installs a Help manual that explains how to use DPSF and includes the full API documentation for all functions and properties of DPSF. The Help manual also provides a number of tutorials, with their source code included in the installer. The Help manual can be found in the DPSF folder in the Windows Start Menu, as well as in the installation directory (typically C:\DPSF\DPSF Help.chm), or it can be viewed online here.

If you require further help and want to post a question, or just want to leave your comments about DPSF, you may do so at the DPSF Forums.
