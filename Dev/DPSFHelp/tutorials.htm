This section contains tutorials showing how to use DPSF to accomplish some different tasks.

 

All of the tutorials have completed source code accompanying them unless otherwise stated.  The tutorial source code can be accessed from the Windows Start Menu -> All Programs -> DPSF -> Tutorials folder, or directly from the installation directory (typically C:\DPSF\Tutorials).

 

It is recommended that you follow along in the tutorial's source code while going through the tutorials as they show you where code goes within the files and functions, giving more context to the tutorials, and will allow you to change and run the code to see how it affects the particle systems. You do not need to perform the actions mentioned in the tutorials on this source code though; it has already been done.  The source code shows what the files should look like once all of the actions in the tutorials has been performed.
